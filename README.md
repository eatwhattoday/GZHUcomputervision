# GZHU Computer Vision Experiment

这是一个实验记录

## 项目概述
该项目包含了一系列的实验，涵盖了从基础图像处理的多个方面。

## 特点
- **实时目标检测**：实现对图像中物体的快速识别。
- **图像分割**：使用先进的算法将图像分割成多个区域或对象。
- **模式识别**：应用机器学习技术来识别图像中的模式和特征。
- **LBP**
- **OTSU**
- **图像检测**
- **图像分割**
- **轮廓绘制**
